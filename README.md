# Chat Application

This is a simple text-based chat application implemented in Python using a client-server model. Two users can exchange messages in real-time through the command line.

## Features

- **Client-Server Model:**
  - Utilizes a client-server architecture for message exchange.

- **Real-time Communication:**
  - Enables real-time communication between two users.

## Usage

1. Run the `server.py` script on one terminal to start the server.
   ```bash
   python server.py
Run the client.py script on separate terminals for each user to connect to the server.

bash
Copy code
python client.py
Enter a unique username for each client when prompted.

Start exchanging messages in real-time.
